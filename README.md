# TP1-OC1
    Para executar: python3 Conversor.py teste.asm no terminal. Basta trocar o nome do arquivo .asm, para o nome do arquivo que você quer executar, por exemplo, para executar instruções do tipo I:

    python Conversor.py codigoI.asm


    -> Ao executar o programa, escolha o tipo de saída desejada:
    -> 1 para exibir o resultado no terminal.
    -> 2 para salvar o resultado em arquivo. Caso escolha essa opção, você precisará informar o nome do arquivo de saída.

    ->Basta mudar a letra a frente da palavra codigo, para a letra do conjunto de instruções correspondente.

    ->Para mudar as bases, basta adicionar hex ao final do comando: python Conversor.py teste.asm hex
